# TaskManager GraphQL

TaskManager GraphQL es una aplicación para la gestión de tareas personales, desarrollada con una arquitectura full stack. El backend utiliza GraphQL con Apollo Server para ofrecer una API flexible y eficiente, mientras que el frontend está construido con React para una experiencia de usuario dinámica. La aplicación incluye autenticación segura con JWT y persistencia de datos en MongoDB
Este es un proyecto Full Stack desarrollado con:

- Node.js
- Express
- GraphQL
- Apollo Server
- MongoDB
- React
- Axios
- Docker

## Características de este proyecto

- Registro de usuarios
- Login con JWT
- Crear tareas
- Obtener tareas
- Backend GraphQL
- Frontend React
- Base de datos MongoDB
- Contenedores Docker

# Pasos para la instalación

## Clonar repositorio

```bash
git clone URL_DEL_REPOSITORIO
```

## Instalar dependencias backend

```bash
npm install
```

## Instalar dependencias frontend

```bash
cd frontend
npm install
```

---

# Ejecutar proyecto con Docker

```bash
docker compose up --build
```
